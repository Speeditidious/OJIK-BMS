# OJIK BMS

BMS 여러 구동기 플레이 데이터를 통합 관리하기 위한 웹 서비스입니다. 효율적인 리딸에 초점이 맞춰져있습니다. 아직 개발 중이며, 현재 베타 테스트도 진행 못하는 수준입니다 (몇몇 기능 동작 확인을 위하여 커밋함).

## Main Features

기본적인 플레이 데이터 시각화 외에도 다음과 같은 기능들을 제공합니다.

- **LR2 기록갱신 날짜 저장**: Beatoraja에는 scorelog.db가 있지만, LR2는 그런 것도 없고 최고기록 달성한 날짜조차 보여주지 않습니다. 서버에 플레이 데이터를 동기화한 날짜를 기준으로 기록갱신한 날짜를 저장하고 시각화에 활용합니다.
- **난이도표 import 및 즐겨찾기**: 개발자가 등록한 난이도표 외에도 따로 등록시킬 수 있습니다. 또한, 관심있는 난이도표들만 즐겨찾기해서 시각화 된 결과를 볼 수 있도록 구현했습니다.
- **(예정) 커스텀 난이도표 및 코스**: 자신만의 난이도표와 코스를 만들고 export할 수 있는 기능입니다.
- **(예정) 없는 차분 업데이트 편의기능**: 동기화된 데이터 기반으로 난이도표 별로 다운로드 받아야하는 것들을 표시하는 기능입니다 (BeMusicSeeker와 비슷한 기능).
- **(예정) 랜덤선곡기**: 유저에 의해 필터링된 pool 내에서 랜덤으로 선곡해주는 기능입니다.
- **(예정) 챗봇**: 유저의 개개인의 플레이데이터를 기반으로 여러 조언(차분 추천 등)을 해주는 챗봇을 개발할 예정입니다.
- **(예정) 차분 시뮬레이터**: 배치 지정해서 원하는 영역 플레이 해볼 수 있는 기능을 추가하려 합니다. 구현하는데 꽤 오래걸릴 것 같아 후순위로 둡니다.

## Coverage

- **지원하는 구동기**: 현재 LR2, Beatoraja 기록만 처리하도록 구현되어있습니다. 추후에 가능하면 Qwilight도 추가할 예정입니다.
- **지원하는 BMS 파일**: `.bms`, `.bme`, `.bml`, `.bmson`, `.pms` (이 외에도 지원해야하는 파일 확장자 있으면 제보 부탁드립니다.)
