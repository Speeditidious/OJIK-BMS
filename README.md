# OJIK BMS

BMS 여러 구동기 플레이 데이터를 통합 관리하기 위한 웹 서비스입니다. 아직 개발 중입니다 (베타 테스트 중).

## How to Use

1. 홈페이지에 접속한다.
2. 클라이언트를 다운로드 받는다.
3. 다운로드 받은 클라이언트 실행 후 동기화 한다 (처음에는 전체 동기화, 그 뒤로 추가한 차분 없으면 빠른 동기화 추천).
4. 홈페이지에 동기화 된 것을 확인한다. ^_^

## Main Features

- **플레이 데이터 시각화**: 데이터베이스를 분석해 플레이 횟수 및 시간 등 스탯, 난이도표 별 클리어 분포, 날짜별 활동 추이 등을 시각화합니다.
- **LR2 기록갱신 날짜 저장**: LR2는 플레이 기록 데이터베이스에 날짜가 저장되지 않아 언제 기록을 달성했는지 추적하기 어려웠습니다. 이제 서버에 기록한 시간을 기반으로 추적할 수 있습니다.
- **경험치 및 레이팅 시스템**: 동기부여를 위해 플레이 기록에 따라 경험치와 레이팅을 부여합니다. 난이도표 별로 다른 공식을 사용하고 있으며, 현재 5KEYS AERY, Satellite, Stella, 발광, 제2발광, 오버조이만 구현되어있습니다. 또한, 같은 레벨 내에 있는 차분들 끼리는 차이를 두지 않았습니다 (Future Work).
- **목표 설정**: 클리어 타입, BP, 판정, 랭크 목표를 설정하고 달성하면 언제 달성했는지 기록할 수 있습니다.

## Coverage

- **지원하는 구동기**: 현재 LR2, Beatoraja 기록만 처리하도록 구현되어있습니다.
